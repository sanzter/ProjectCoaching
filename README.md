Pemesanan Kue Pada Toku Kue Sejahtera

Project ini bertujuan agar pemesan dapat memesan kue secara online maupun aplikasi offline berbasis CLI

Install Instruction
1. Download project.java
2. Open file yang telah di download menggunakan VSCode / Eclipse For Java
3. Jika menggunakan VSCode download Extension Pack For Java. Untuk Eclipse disarankan menggunakan java 21 ke atas untuk run file ini
4. Setelah membuka file, file dapat langsung di run.
